# Camera Relative Pose From Two Panorama
Required:
------------------
- Visual Studio >= 2019 (recommended)

Getting the project
------------------

1. Run `GenerateProjects.bat` to generate project files for Visual Studio 2019

2. Run `PanoramicCameraPose.sln` and build
